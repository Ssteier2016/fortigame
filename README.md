# Fortigame
Juego 2D multijugador de combate, recolección e intercambio de cartas, ambientado en un mapa realista de CABA (Palermo).

## Tecnologías
- Frontend: Phaser.js
- Backend: Node.js, Express, Socket.io
- Base de datos: MongoDB
- Despliegue: Render
